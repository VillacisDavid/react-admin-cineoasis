# Admin - Cine Oasis
Frontend para página de administración de funciones para el cine ficticio **Cine Oasis**  
#### Especificación del proyecto
Se utilizan las siguientes herramientas:  

    - React  
    - Vite  
    - NodeJS  
    - Tailwind

#### Por hacer

    - Obtener datos desde API
    - Pulír la presentación
